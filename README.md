# DropSpot

DropSpot is a location-based social map where users can leave notes tied to real places, discover nearby activity, and reveal full note content only when they are physically close enough.

## Features

- Create location-based notes with a title and description
- Explore notes anywhere on the map
- Unlock note content only when physically nearby
- Automatic hotspot detection when multiple notes appear in the same area
- Clustered map markers for dense areas
- Manage your own notes, including editing and deleting
- Relative timestamps such as "2 hours ago"

## Tech Stack

- Frontend: HTML, CSS, JavaScript, Leaflet
- Backend: Flask, Flask-CORS
- Database: MySQL

## Running the Project

Start the backend from the project root:

python3 backend/app.py
